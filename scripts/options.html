URL Format

Use the 'add address' bar to enter the names of any website you wish to block.

When entering a website name, follow the format www.example.com or www.example.com/path.

Do not include any protocol prefix, such as:

  • http://
  • https://

Do not include a single '/' at the end.

Valid examples include:

  • www.bbc.co.uk
  • twitter.com/home
  • www.youtube.com/watch?v=dQw4w9WgXcQ

Reset to Default

To clear all stored values and reset all values to default, in the 'add address' bar, enter --clear and press 'Add'.
